Sobre el Proyecto

Este proyecto utiliza Python y hardware interactivo para codificar y decodificar mensajes en código Morse, combinando señales visuales y sonoras para aprendizaje práctico.

Objetivo

Demostrar cómo la tecnología y la programación pueden facilitar el aprendizaje de sistemas de comunicación clásicos, como el código Morse.

Tecnologías Aplicadas

Python

Lenguaje de programación para procesar y codificar los mensajes Morse.

Raspberry Pi

Microcomputadora que controla los LEDs y buzzer para señales Morse.

LED

Señales visuales para representar los puntos y rayas del código Morse.

Buzzer

Señales sonoras para reproducir el código Morse audible.
